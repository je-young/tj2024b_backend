[컴파일] : 컴퓨터가 이해할 수 있는 언언로 번역하는 프로그램, 컴파일 : 번역하는 작업
[자바 장점]
  1. 운영체제(플랫폼)으로 부터 독립적이다. 코드(.class)를 한번 작성하면 여러 운영체제에서 실행활 수 있다.
  2. 객체지향언어(상호관계), 유지보수, 확장성이 좋다.
  3. 가비지 컬렉터(GC : ) : 메모리를 자동(동적)으로 관리
  4. 오픈소스(JDK, SPRING, JSP) : 남이 만들어둔 소스를 공개적으로 사용가능하다.

[자바 사용처] : 웹, 앱, 게임

[JDK 자바설치], 버전의 중요성은 없다, JDK8 이상이면 강의 듣기 가능합니다.

[JDK 설치확인]
  1. 터미널
  2. java -version

[이클립스 설치]
  - IDE(통합개발환경) : 코드를 작성하는데 여러가지 기능 제공하는 프로그램
    - 1. vscode, 이클립스 : IDE(통합개발환경): 코드를 작성하는데 여러가지 기능 제공하는 프로그램
    - 2. vscode : (무료) 프론트엔드 개발에서 주로 사용
    - 3. 이클립스 : (무료)백엔드 개발에서 주로 사용
    - 4. 인텔리제이 : (부분유료) 프론트엔드, 백엔드 회사에서 주로 사용
